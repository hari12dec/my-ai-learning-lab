## Library Used
      - Pandas 
          -> A powerful Python library for working with structured data like tables, CSVs, and Excel files.
            It helps you clean, analyze, filter, and manipulate data easily using DataFrames.
      - Sklearn
          -> A Python library used for building machine learning models.
            It provides tools for training, testing, and evaluating algorithms easily.
      - Pickle
          -> A Python module for saving objects to a file.
            Used to store ML models and load them later without retraining.
